[[# 🛒 Customer Shopping Behavior Analysis (End-to-End Project)

## 📌 Project Overview
This project analyzes customer shopping behavior to understand purchasing patterns, sales trends, and product performance. The analysis was performed using **Python, SQL, and Power BI**, creating an end-to-end data analytics workflow from data extraction to interactive dashboard visualization.

The goal of this project is to generate actionable insights that help businesses better understand customer behavior and improve decision-making.

---

## 🎯 Objectives

- Analyze customer purchasing patterns
- Identify sales trends and product performance
- Extract and process data using SQL
- Perform data cleaning and analysis using Python
- Build an interactive dashboard for business insights

---

## 🛠️ Tools & Technologies

- **Python (Pandas, NumPy)** – Data cleaning and analysis  
- **SQL** – Data extraction and querying  
- **Power BI** – Dashboard creation and data visualization  
- **Jupyter Notebook** – Analysis environment  

---

## 🔎 Project Workflow

### 1️⃣ Data Extraction (SQL)
- Extracted relevant customer transaction data using SQL queries
- Filtered and processed datasets for analysis

### 2️⃣ Data Cleaning & Processing (Python)
- Cleaned and analyzed customer shopping data using **Pandas and NumPy**
- Handled missing values and removed duplicate records
- Performed data transformation for analysis

### 3️⃣ Exploratory Data Analysis (EDA)
- Analyzed customer purchase behavior
- Identified sales trends and product demand
- Examined purchasing frequency and spending patterns

### 4️⃣ Data Visualization (Power BI)
- Built an **interactive Power BI dashboard**
- Visualized customer insights
- Analyzed product performance
- Displayed sales trends and category performance

---

## 📊 Key Insights

- Identified top-selling product categories
- Observed patterns in customer purchasing behavior
- Analyzed sales distribution across product segments
- Highlighted customer spending trends
